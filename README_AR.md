# بوت ستريميو ديسكورد الاحترافي (Stremio Discord Selfbot) 🎬🍿

بوت متطور لبث محتوى Stremio مباشرة داخل غرف الديسكورد باستخدام تقنية Go-Live، مصمم لتقديم تجربة سينمائية مستقرة وبجودة عالية.

---

## 🌟 المميزات الرئيسية

- **دعم كامل للمسلسلات**: إمكانية اختيار الموسم والحلقة عبر أوامر سهلة (`$s` و `$e`).
- **بث احترافي (Pro Streaming)**: معدل نقل بيانات **10Mbps** وتحديث **30fps** مع تفعيل تسريع العتاد (Hardware Acceleration).
- **نظام ترجمة متطور**: دعم الترجمة المدمجة (Hardcoded) بترميز `libass` لضمان ظهور الترجمة العربية بوضوح وتنسيق ممتاز.
- **تجديد تلقائي للروابط (Auto-Refresh)**: ميزة فريدة تقوم بتحديث روابط البث المنتهية قبل بدء التشغيل لمنع انهيار البث (Crash).
- **مزامنة ذكية (Deep Sync)**: استخدام تقنيات `+genpts` و `aresample` لضمان عدم انفصال الصوت عن الصورة.
- **متابعة المشاهدة (Resume)**: حفظ تلقائي لمكان توقفك في الفلم أو المسلسل لضمان العودة للمشاهدة لاحقاً من نفس النقطة.

---

## 🏗 المعمارية التقنية (Architecture)

يعمل البوت كجسر تقني يربط بين إضافات Stremio وبروتوكول ديسكورد للبث:

1.  **محاكي المحرك (Engine Mimicry)**: يقوم البوت بمحاكاة سلوك تطبيق Stremio الرسمي من حيث إرسال رؤوس الطلبات (Headers) والـ Proxy Headers لضمان عمل الروابط المشفرة.
2.  **معالج البث (Stream Orchestrator)**: منطق برمجي يقوم بفرز الجودات واختيار أفضل مصدر متاح.
3.  **تشفير FFmpeg اللحظي**:
    -   **تغيير الدقة (Scaling)**: معالجة الصورة بدقة 720p أو 1080p بطريقة Bicubic لسرعة الأداء.
    -   **دمج الفلاتر**: دمج فلاتر الصورة والترجمة في مسار واحد (Pipeline) لتوفير موارد الجهاز.
4.  **نظام الإشعارات المباشرة**: تحديث المستخدم بحالة معالجة الطلب (Applying subtitles, Switching audio).

## 🛠 متطلبات التطوير المستقبلية (Roadmap)

المشروع مفتوح للتطوير والتحسين في المجالات التالية:
- **لوحة تحكم (Dashboard)**: إنشاء واجهة ويب (Vite/Next.js) للتحكم في البوت بدلاً من الأوامر النصية.
- **دعم سيرفرات متعددة**: تطوير النظام ليدعم العمل في أكثر من سيرفر في نفس الوقت بشكل مستقل تماماً.
- **مزامنة حساب Stremio**: ربط البوت بحساب المستخدم الرسمي لجلب قائمة "المتابعة" (Continue Watching) وقائمة الأفلام المفضلة.
- **تحسين استجابة البحث**: استخدام تقنيات الـ Caching لنتائج البحث لتقليل وقت الاستجابة.

## 🚀 التشغيل السريع
1.  تثبيت FFmpeg مع دعم `libass`.
2.  تثبيت المكتبات: `npm install`
3.  إضافة الـ Token في ملف `.env`.
4.  البناء والتشغيل: `npm run build && npm start`
