# بوت ستريميو ديسكورد الاحترافي (Stremio Discord Selfbot) 🎬🍿

## 🌟 المميزات الرئيسية

- **دعم كامل للمسلسلات**: إمكانية اختيار الموسم والحلقة عبر أوامر سهلة (`$s` و `$e`).
- **بث احترافي (Pro Streaming)**: معدل نقل بيانات **10Mbps** وتحديث **30fps** مع تفعيل تسريع العتاد.
- **نظام ترجمة متطور**: دعم الترجمة المدمجة (Hardcoded) بترميز `libass` لضمان ظهور الترجمة العربية بوضوح وتنسيق ممتاز.
- **تجديد تلقائي للروابط (Auto-Refresh)**: ميزة فريدة تقوم بتحديث روابط البث المنتهية قبل بدء التشغيل لمنع انهيار البث.
- **متابعة المشاهدة (Resume)**: حفظ تلقائي لمكان توقفك في الفلم أو المسلسل لضمان العودة للمشاهدة لاحقاً من نفس النقطة.

## 🏗 المعمارية التقنية (Architecture)

يعمل البوت كجسر تقني يربط بين إضافات Stremio وبروتوكول ديسكورد:

1.  **محاكي المحرك (Engine Mimicry)**: يقوم البوت بمحاكاة سلوك تطبيق Stremio الرسمي لضمان عمل الروابط المشفرة.
2.  **معالج البث (Stream Orchestrator)**: منطق برمجي يقوم بفرز الجودات واختيار أفضل مصدر متاح.
3.  **تشفير FFmpeg اللحظي**: معالجة الصورة والترجمة في مسار واحد لتوفير موارد الجهاز وضمان التزامن.

## 📘 دليل الإعداد خطوة بخطوة

### 1. المتطلبات الأساسية
- **Node.js** إصدار 18 أو أحدث.
- **FFmpeg**: يجب أن يكون مثبتاً مع دعم مكتبة `libass` (ضروري للترجمة).
- **Stremio Streaming Server**: يجب وجود نسخة تعمل من محرك ستريميو (غالباً على المنفذ 11470)؛ هذا **شرط أساسي** لترجمة روابط البث.
  - **Mac**: `brew tap homebrew-ffmpeg/ffmpeg && brew install homebrew-ffmpeg/ffmpeg/ffmpeg --with-libass`

### 2. الإعداد البرمجي
1.  **تحميل المشروع**: `git clone https://github.com/3-pr/stremio-discord`
2.  **تثبيت المكتبات**: `npm install`
3.  **إعداد المتغيرات**: أنشئ ملف `.env` وضع فيه بياناتك:
    ```env
    SELFBOT_TOKEN=your_discord_token_here
    OWNER_ID=your_discord_id
    COMMAND_PREFIX=$
    STREMIO_HOST=localhost
    STREMIO_PORT=11470
    ```

### 3. التشغيل
1.  **بناء المشروع**: `npm run build`
2.  **البدء**: `npm start`
3.  انضم لقناة صوتية واكتب `$play <movie name>` للبدء.
