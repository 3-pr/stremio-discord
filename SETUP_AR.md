# دليل إعداد وتشغيل البوت لـ Stremio 🍿

هذا الملف يشرح لك خطوة بخطوة كيفية إعداد وتشغيل بوت ديسكورد لمشاركة بث Stremio مع أصدقائك باللغة العربية.

---

## ⚠️ تحذير هام جداً
هذا المشروع يستخدم نظام "Selfbot" (استخدام حساب مستخدم عادي كـ بوت لمشاركة شاشة اللعب Go Live)، وهو أمر **مخالف كلياً لشروط استخدام ديسكورد (ToS)**. 
استخدام أدوات الأتمتة على حسابك الأساسي قد يؤدي إلى **حظر دائم للحساب (Banned)**.
**يُنصح بشدة وبشكل قطعي استخدام حساب ديسكورد وهمي (Burner Account) لتشغيل هذا البوت ولتجنب المخاطر.**

---

## 🛠 المتطلبات الأساسية
- تحميل وتثبيت **Node.js** (الإصدار 20 أو أحدث).
- تحميل برنامج **FFmpeg** للتشفير، والتأكد من توافر مكتبات `libzmq`.
- تواجد سيرفر Stremio يعمل محلياً (`stremio-service`).

---

## 🎬 الخطوة الأولى: إعداد سيرفر ستريميو
لكي يتمكن البوت من جلب الأفلام وإرسالها، يجب أن يعمل سيرفر ستريميو بجهازك محلياً بشكل صحيح.

1. **تشغيل Stremio Server**: تأكد من عمل السيرفر على بورت `11470`. 
2. **إضافة الإضافات (Add-ons)**: إذا كان السيرفر محلياً، قم بتسجيل الدخول بأي حساب Stremio يمتلك إضافات ممتازة مثل **Torrentio** (للأفلام) و **OpenSubtitles** (للترجمة)، لأن البوت سيتعرف عليها ويستخدمها تلقائياً.

---

## 🗝 الخطوة الثانية: استخراج التوكن الخاص بحساب الديسكورد
> **تنبيه أمني:** لا تشارك التوكن الخاص بك أبداً. من يحصل عليه يمكنه اختراق حسابك بالكامل وإرسال رسائل باسمك.

1. **طريقة الحصول على توكن الحساب (Selfbot Token)**:
   - افتح تطبيق ديسكورد من خلال المتصفح (Chrome/Edge/Brave).
   - اضغط على زر `F12` لفتح نافذة المطورين "Developer Tools".
   - انتقل إلى نافذة/تبويب `Network` (أو الشبكة).
   - اكتب أي رسالة وأرسلها في أي شات.
   - ستظهر لك قائمة طلبات في نافذة الشبكة، ابحث عن طلب اسمه `messages` واضغط عليه.
   - في القائمة الجانبية، انزل للأسفل لقسم `Request Headers` وابحث عن كلمة `authorization`.
   - النص الطويل والمشفّر بجانبها هو **التوكن** الخاص بالحساب، قم بنسخه بحذر.

2. **طريقة الحصول على معرفك الخاص (Owner ID)**:
   - البوت مبرمج لكي يستجيب لأوامرك أنت فقط.
   - فعل وضع المطورين في إعدادات الديسكورد (Settings > Advanced > Developer Mode).
   - اضغط بالزر الأيمن (Right-Click) على حسابك الشخصي واختر `Copy User ID`.

## 📦 الخطوة الثالثة: طريقة التثبيت والتشغيل المحلي
1. في مجلد هذا المشروع المليء بالملفات، ابحث عن ملف يسمى `.env` (مخفي عادة أو اسمه `.env.example`).
2. قم بتعريف المتغيرات وضع البيانات كالتالي:
```env
SELFBOT_TOKEN=الصق_التوكن_حقك_هنا
OWNER_ID=الصق_ايدي_الديسكورد_تبعك_هنا
COMMAND_PREFIX=$
STREMIO_HOST=localhost
STREMIO_PORT=11470
```
3. افتح موجه الأوامر (Terminal / CMD) وتأكد أنك داخل مسار مجلد المشروع، ثم اكتب الأوامر التالية بالترتيب:

- لتحميل المكتبات والمتطلبات الخاصة بالبرمجة:
```bash
npm install
```
- لبناء الكود إلى لغة الجافا سكريبت:
```bash
npm run build
```
- لتشغيل البوت فعلياً:
```bash
npm start
```

---

## 🐳 طريقة أخرى سهلة: التثبيت ببرنامج Docker
لمن يملكون `Docker`، هذه أسهل وأكثر الطرق أماناً بدلاً من تثبيت Stremio يدوياً، افتح التيرمنال واكتب فقط:
```bash
docker-compose up -d --build
```
هذا الكود سيقوم بتشغيل سيرفر Stremio داخلي مع البوت في بيئة معزولة ونظيفة تلقائياً.

## 📜 قائمة الأوامر
هذه الأوامر يجب أن تكتبها بحسابك أنت الشخصي داخل ديسكورد كرسالة عادية، وسوف يستجيب لك البوت (بدون استخدام أوامر Slash):

- `$play <اسم الفيلم أو المسلسل>` : سيقوم بالبحث وعرض جودات وإصدارات الفيلم. بانتظار ردك لاختيار النسخة.
- `$select <الرقم>` : استخدمه مباشرة بعد أمر `play` لاختيار التورنت ورقم الجودة حتى يبدأ البث المباشر.
- `$play-id <imdb>` : طريقة مباشرة لتشغيل الفيلم عبر رمز الفيلم الرسمي (IMDb).
- `$stop` : إيقاف البث بالكامل وطرد البوت من الغرفة الصوتية.
- `$pause` : لإيقاف الفيلم مؤقتاً (ستتوقف الشاشة).
- `$resume` : لاستئناف عرض الفيلم (سيأخذ ثانيتين لاسترجاع المشهد).
- `$subtitles` : لعرض وطباعة الترجمات المتوفرة (من إضافاتك مثل أوبن سبتايتلز). يمكنك الرد برقم الترجمة لتفعيلها.
- `$queue add <اسم الفيلم>` : لإضافة فيلم أو حلقة لقائمة الانتظار وعرضها تلقائياً متى انتهى العرض الحالي.
- `$skip` : لتجاوز العرض الحالي والانتقال لما يليه في القائمة فوراً.
- `$volume <0-100>` : لرفع أو خفض صوت الفيلم الذي يصل لأصدقائك في الغرفة.
- `$info` : لاستعراض معلومات ما الذي يتم عرضه حالياً بدقة.

بالتوفيق بالمشاهدة الممتعة مع أصدقائك عبر الديسكورد! 🎉
